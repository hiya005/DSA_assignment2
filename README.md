# Data Structures using Python

This project contains basic implementations of important data structures using Python.
All programs are written in a simple and easy-to-understand way.

---

## 📌 Contents

The project includes:

- Dynamic Array  
- Singly Linked List  
- Doubly Linked List  
- Stack using Linked List  
- Queue using Linked List  
- Parenthesis Checker (using Stack concept)  

---

## 🧠 Description

### 1. Dynamic Array
- Automatically resizes when full  
- Supports append, pop, indexing  
- Built using ctypes  

### 2. Singly Linked List
- Each node stores data and next pointer  
- Operations:
  - Insert at end  
  - Delete by value  
  - Traverse  

### 3. Doubly Linked List
- Each node has previous and next pointers  
- Allows forward and backward movement  
- Operations:
  - Insert after a node  
  - Delete by position  

### 4. Stack (Using Linked List)
- Follows LIFO (Last In First Out)  
- Operations:
  - Push  
  - Pop  
  - Peek  

### 5. Queue (Using Linked List)
- Follows FIFO (First In First Out)  
- Operations:
  - Enqueue  
  - Dequeue  

### 6. Parenthesis Checker
- Uses stack concept  
- Checks if brackets are balanced
